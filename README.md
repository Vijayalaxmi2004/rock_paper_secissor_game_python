# rock_paper_secissor_game_python

Simple Rock-Paper-Scissors terminal game written in Python.

## Features
- Play one round against the computer
- Validates user input (Rock, Paper, Scissors)

## Requirements
- Python 3.8+

## Usage
Run the game from the repository root:

```bash
python3 main.py
```

Type `Rock`, `Paper`, or `Scissors` when prompted (case-insensitive).

## Example
```
Enter your choice (Rock, Paper, Scissors): rock
Computer chose: Paper
you lose
```
